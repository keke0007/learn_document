# Flink SQL 学习目录

本目录专注 **Flink SQL**（含 Table API/SQL 核心概念与生产常用模式）。

推荐阅读顺序：
- `GUIDE.md`：Flink SQL 费曼学习指南（知识点 + 案例 + 验证数据 + 总结清单）
- `cases/`：拆分后的案例文档（可复制到 SQL Client 执行）
- `data/`：配套验证数据（小而可算，便于对照输出）
- `scripts/`：运行说明与常用命令（SQL Client）
