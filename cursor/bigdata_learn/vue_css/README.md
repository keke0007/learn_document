# Vue + CSS 学习目录

本目录专注 **Vue 页面布局思路** 与 **CSS 核心知识点**（含现代布局方案与最佳实践）。

推荐阅读顺序：
- `GUIDE.md`：Vue 布局 + CSS 费曼学习指南（知识点 + 案例 + 验证文件 + 总结清单）
- `cases/`：拆分后的案例文档（Vue 组件 + CSS 样式）
- `data/`：示例 HTML/Vue 文件（可直接在浏览器打开验证）
- `scripts/`：运行说明与常用命令

案例索引：
- `cases/README.md`：案例总览索引页
- `cases/01_vue_layout_basic_structure.md`：Vue 页面布局基本思路（组件化 + Flexbox/Grid）
- `cases/02_css_flexbox_grid.md`：CSS Flexbox 与 Grid（现代布局核心）
- `cases/03_css_positioning_spacing.md`：CSS 定位与间距（Position / Margin / Padding）
- `cases/04_vue_scoped_styles_bem.md`：Vue 样式作用域与命名规范（Scoped / BEM）

目录结构：
```
vue_css/
├── README.md
├── GUIDE.md
├── cases/
├── data/
└── scripts/
```
