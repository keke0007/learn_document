# Spark SQL 学习目录

本目录专注 **Spark SQL**（含 DataFrame / Dataset / SQL 及常见调优实践）。

推荐阅读顺序：
- `GUIDE.md`：Spark SQL 费曼学习指南（知识点 + 案例 + 验证数据 + 总结清单）
- `cases/`：拆分后的案例文档（可在 `spark-shell` / `spark-sql` / 应用代码中执行）
- `data/`：配套验证数据（小而可算，便于对照输出）
- `scripts/`：运行说明与常用命令

目录结构：
```
spark_sql/
├── README.md
├── GUIDE.md
├── cases/
├── data/
└── scripts/
```
