# Java 数据结构与算法学习目录

本目录专注 **基于 Java 的数据结构与算法** 学习（偏实战与面试常考），配套小数据集与可运行示例。

推荐阅读顺序：
- `GUIDE.md`：数据结构与算法费曼学习指南（知识点 + 案例 + 验证数据 + 总结清单）
- `cases/`：拆分后的案例文档（Java 代码 + 思路讲解）
- `data/`：配套验证数据（小而可算，便于手算/断点调试）
- `scripts/`：运行说明与示例代码入口

目录结构：
```
data_structure/
├── README.md
├── GUIDE.md
├── cases/
├── data/
└── scripts/
```
