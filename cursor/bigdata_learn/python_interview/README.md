## 1. Python 核心

### 1.1 装饰器

**装饰器定义**
- 装饰器是修改或增强函数功能的函数
- 使用 @ 语法糖
- 可以带参数

**常见装饰器**
- @property：属性装饰器
- @staticmethod：静态方法
- @classmethod：类方法
- @functools.wraps：保持函数元信息

### 1.2 生成器和迭代器

**生成器**
- 使用 yield 关键字
- 延迟计算，节省内存
- 适合处理大数据量

**迭代器协议**
- `__iter__()` 方法
- `__next__()` 方法
- 实现迭代器协议

### 1.3 上下文管理器

**with 语句**
- 资源管理
- 异常处理
- 自动清理

**实现方式**
- `__enter__()` 和 `__exit__()` 方法
- contextlib.contextmanager 装饰器

### 1.4 元类

**元类定义**
- 类的类
- 控制类的创建
- 高级特性

**应用场景**
- 单例模式
- 类注册
- 框架开发

## 2. 数据结构与算法

### 2.1 数据结构性能

**列表（List）**
- 末尾添加：O(1)
- 开头插入：O(n)
- 随机访问：O(1)

**字典（Dict）**
- 查找：O(1) 平均
- 插入：O(1) 平均
- 删除：O(1) 平均

**集合（Set）**
- 查找：O(1) 平均
- 交集：O(min(m, n))
- 并集：O(m + n)

### 2.2 排序算法

**快速排序**
- 平均时间复杂度：O(n log n)
- 最坏情况：O(n²)
- 不稳定排序

**归并排序**
- 时间复杂度：O(n log n)
- 稳定排序
- 需要额外空间

**堆排序**
- 时间复杂度：O(n log n)
- 原地排序
- 不稳定排序

### 2.3 查找算法

**二分查找**
- 时间复杂度：O(log n)
- 要求有序数组
- 分治思想

**哈希查找**
- 时间复杂度：O(1) 平均
- 使用字典实现
- 空间换时间

### 2.4 动态规划

**特点**
- 重叠子问题
- 最优子结构
- 自底向上求解

**经典问题**
- 斐波那契数列
- 0-1背包问题
- 最长公共子序列
