性能测试数据记录
==================

测试环境：
- CPU: Intel i7-9700K
- 内存: 16GB DDR4
- JDK: OpenJDK 11
- OS: Windows 10

测试1：ArrayList vs LinkedList
------------------------------
操作：头部插入 100000 个元素
ArrayList: 5234ms
LinkedList: 48ms
结论：LinkedList 在头部插入操作上性能明显优于 ArrayList

测试2：HashMap vs TreeMap
-------------------------
操作：插入 100000 个键值对
HashMap: 156ms
TreeMap: 523ms
结论：HashMap 在插入操作上性能优于 TreeMap

测试3：并发测试
--------------
场景：10 个线程同时操作 HashMap
HashMap（线程不安全）: 最终大小不一致，存在数据丢失
ConcurrentHashMap（线程安全）: 最终大小正确，无数据丢失

测试4：GC 性能测试
-----------------
堆大小：512MB
对象数量：1000000
GC 次数：15
GC 总耗时：234ms
平均 GC 耗时：15.6ms
