# Dify 环境搭建与配置要点（学习用）

> 说明：Dify 的安装方式可能随版本变化。本文件只给“学习要点”，具体命令请以官方文档为准。

## 1. 常见部署方式

- Docker / Docker Compose（本地学习最常见）
- 云端部署（生产环境需考虑网络、存储、权限与审计）

## 2. 关键配置清单（学习时最常用）

- **模型供应商配置**：API Key、模型名称、超时与重试
- **向量库/检索配置**：用于知识库（RAG）的索引与检索
- **鉴权与密钥**：工具调用的 API Key、应用 API Key（只给最小权限）

## 3. 学习建议

1. 先不接复杂外部系统，优先跑通：
   - Chat 应用 → RAG → Workflow + Tool
2. 每次只改一个变量（Prompt 或 RAG 参数），用 `data/sample_queries.json` 回归验证。
