# 订单与售后规则（示例）

## 1. 退款与退货

- **未发货订单**：可申请取消订单并退款，通常 1-3 个工作日原路退回。
- **已发货订单**：如需退货，需在签收后 7 天内申请；商品需保持完好，不影响二次销售。
- **虚拟商品**：发放后不支持无理由退货（质量问题除外）。

## 2. 售后范围

- **质量问题**：提供照片/视频证明后可优先处理换货或退款。
- **非质量问题**：按“7 天无理由”规则处理（不适用虚拟商品/定制类商品）。

## 3. 运费说明

- **质量问题退货**：运费由商家承担（需按指引寄回）。
- **非质量问题退货**：运费由用户承担。

## 4. 注意事项

- 退款到账时间以支付渠道为准，通常 1-3 个工作日。
- 超过售后时限的订单需转人工客服进一步确认。
