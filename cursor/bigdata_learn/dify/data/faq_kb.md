# 客服 FAQ（示例）

## 1. 发货与物流

- **Q：多久发货？**  
  通常在付款后 24 小时内发货（节假日顺延）。

- **Q：如何查询物流？**  
  在订单详情页查看物流单号；如无单号，可能尚未出库。

## 2. 发票

- **Q：可以开发票吗？**  
  支持开具电子发票。下单后可在订单详情申请。

- **Q：发票抬头能修改吗？**  
  发票一旦开具不可修改，请提交前确认信息。

## 3. 账号与隐私

- **Q：如何修改绑定手机号？**  
  在“账号设置-安全”中修改，需短信验证。

- **Q：客服能看到我的完整手机号吗？**  
  客服仅能看到脱敏信息，无法查看完整手机号与支付信息。
