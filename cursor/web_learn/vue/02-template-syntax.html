2. 表达式

数字：{{ number }}

计算：{{ number + 10 }}

三元运算符：{{ number > 10 ? '大于10' : '小于等于10' }}

方法调用：{{ message.toUpperCase() }}

3. 属性绑定

ID 绑定：动态 ID

Class 绑定：动态 Class

4. 指令

v-if 指令：这段文字可以显示/隐藏

v-show 指令：这段文字可以显示/隐藏

5. 修饰符

输入值：{{ trimmedText }}

数字值：{{ numberValue }} (类型: {{ typeof numberValue }})

6. 缩写

v-bind 缩写：:id="value" 等同于 v-bind:id="value"

v-on 缩写：@click="handler" 等同于 v-on:click="handler"
