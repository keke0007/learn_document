# Vue.js 学习案例集合

本文件夹包含 Vue.js 的学习案例与示例代码。

## 📚 学习案例列表

### 基础篇
1. **01-vue-introduction.html** - Vue 简介和基础使用
2. **02-template-syntax.html** - 模板语法
3. **03-data-binding.html** - 数据绑定
4. **04-directives.html** - 指令（v-if, v-for, v-show 等）

### 响应式数据篇
5. **05-computed-properties.html** - 计算属性
6. **06-watchers.html** - 侦听器
7. **07-methods.html** - 方法

### 事件处理篇
8. **08-event-handling.html** - 事件处理
9. **09-form-binding.html** - 表单绑定

### 组件篇
10. **10-components-basic.html** - 组件基础
11. **11-component-props.html** - 组件 Props
12. **12-component-events.html** - 组件事件
13. **13-component-slots.html** - 插槽

### 生命周期篇
14. **14-lifecycle-hooks.html** - 生命周期钩子

### 高级特性篇
15. **15-conditional-rendering.html** - 条件渲染
16. **16-list-rendering.html** - 列表渲染
17. **17-class-style-binding.html** - Class 和 Style 绑定
18. **18-refs.html** - 模板引用

### 实际应用篇
19. **19-todo-app.html** - 待办事项应用
20. **20-counter-app.html** - 计数器应用
21. **21-user-management.html** - 用户管理
22. **22-shopping-cart.html** - 购物车

## 🎯 学习建议

1. 按顺序学习，从基础到高级
2. 每个案例都包含详细注释和说明
3. 打开浏览器查看效果
4. 尝试修改代码，观察响应式变化
5. 结合 Vue DevTools 调试

## 📝 使用说明

1. 直接在浏览器中打开 HTML 文件
2. 所有案例使用 Vue 3 CDN 版本，无需安装
3. 打开浏览器开发者工具查看 Vue DevTools
4. 修改数据观察响应式更新

## 🔧 技术栈

- **Vue 3** - 使用 CDN 引入
- **现代浏览器** - Chrome、Firefox、Edge 等
- **Vue DevTools** - 浏览器扩展（推荐安装）

## 📖 学习路径

1. **入门阶段**：案例 1-4（基础语法和指令）
2. **进阶阶段**：案例 5-9（响应式数据和事件）
3. **组件阶段**：案例 10-13（组件开发）
4. **高级阶段**：案例 14-18（高级特性）
5. **实战阶段**：案例 19-22（实际项目）

## 💡 提示

- 所有案例都是独立的 HTML 文件，可以直接运行
- 使用 Vue 3 Composition API 和 Options API 两种写法
- 代码中包含详细的中文注释
- 每个案例都有可交互的演示
