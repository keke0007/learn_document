过渡属性总结

  • transition-property - 要过渡的属性 (all, color, width 等)
  • transition-duration - 过渡持续时间 (0.3s, 500ms 等)
  • transition-timing-function - 时间函数
    • ease - 慢-快-慢（默认）
    • ease-in - 慢-快
    • ease-out - 快-慢
    • ease-in-out - 慢-快-慢（对称）
    • linear - 匀速
  • transition-delay - 过渡延迟时间
  • transition - 简写属性 (property duration timing-function delay)
