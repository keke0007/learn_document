Grid 常用属性总结

  • display: grid - 启用 Grid
  • grid-template-columns - 定义列 (1fr, repeat(), auto-fit)
  • grid-template-rows - 定义行
  • gap - 网格间距
  • grid-column - 项目跨列 (1 / -1, span 2)
  • grid-row - 项目跨行
  • grid-template-areas - 定义区域
  • justify-content - 水平对齐
  • align-content - 垂直对齐
