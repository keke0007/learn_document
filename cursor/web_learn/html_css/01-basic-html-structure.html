HTML 基础结构

1. HTML 文档结构

每个 HTML 文档都包含以下基本结构：

<!DOCTYPE html>          
<html>                 
  <head>               
    <title>标题</title>
  </head>
  <body>               
    内容
  </body>
</html>

2. 常用标签

  • <h1> 到 <h6> - 标题标签
  • <p> - 段落
  • <div> - 块级容器
  • <span> - 行内容器
  • <a> - 链接
  • <img> - 图片
  • <ul>, <ol>, <li> - 列表

3. 语义化标签（HTML5）

  • <header> - 页眉
  • <nav> - 导航
  • <main> - 主要内容
  • <article> - 文章
  • <section> - 章节
  • <aside> - 侧边栏
  • <footer> - 页脚

4. 示例：语义化结构

<header>
  <h1>网站标题</h1>
  <nav>导航菜单</nav>
</header>

<main>
  <article>
    <h2>文章标题</h2>
    <p>文章内容...</p>
  </article>
</main>

<footer>
  <p>版权信息</p>
</footer>

提示：使用语义化标签可以让代码更易读，也有利于 SEO 和可访问性。
