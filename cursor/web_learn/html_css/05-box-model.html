CSS 盒模型案例

1. 标准盒模型结构

盒模型组成：

  • Content（内容） - 实际内容区域
  • Padding（内边距） - 内容与边框之间的空间
  • Border（边框） - 围绕内边距的边框
  • Margin（外边距） - 元素与其他元素之间的空间
标准盒模型示例

2. box-sizing 属性对比

content-box (默认)
总宽度 = width + padding + border
border-box
总宽度 = width (包含 padding 和 border)

推荐：使用 box-sizing: border-box 可以更直观地控制元素大小。

3. 外边距 (Margin)

正常外边距
上外边距增大
下外边距增大
左外边距增大
右外边距增大
使用 margin: auto 居中

4. 内边距 (Padding)

小内边距 (10px)
中等内边距 (20px)
大内边距 (40px)
不对称内边距 (上10px 右30px 下20px 左40px)

5. 边框样式

实线边框 (solid)
虚线边框 (dashed)
点线边框 (dotted)
双线边框 (double)
圆角边框

盒模型计算公式

content-box (默认)：

总宽度 = width + padding-left + padding-right + border-left + border-right

总高度 = height + padding-top + padding-bottom + border-top + border-bottom


border-box：

总宽度 = width (已包含 padding 和 border)

总高度 = height (已包含 padding 和 border)
