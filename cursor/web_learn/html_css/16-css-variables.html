CSS 变量 (Custom Properties) 案例

1. 基础 CSS 变量使用

CSS 变量使用 --变量名 定义，通过 var(--变量名) 使用。

:root {
  --primary-color: #667eea;
  --spacing-medium: 20px;
}

.card {
  background: var(--primary-color);
  padding: var(--spacing-medium);
}

CSS 变量的优势

  • 集中管理 - 所有样式值集中定义，易于维护
  • 动态修改 - 可以通过 JavaScript 动态改变
  • 作用域 - 支持局部覆盖，实现主题切换
  • 继承 - 变量可以继承和覆盖
  • 回退值 - 支持默认值：var(--color, #000)
