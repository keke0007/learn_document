动画属性总结

  • transition - 过渡效果 (property duration timing-function)
  • @keyframes - 定义关键帧动画
  • animation - 应用动画 (name duration timing-function delay iteration)
  • animation-delay - 动画延迟
  • animation-iteration-count - 重复次数 (infinite)
  • animation-direction - 方向 (normal, reverse, alternate)
  • animation-fill-mode - 填充模式 (forwards, backwards)
