Flexbox 常用属性总结

  • display: flex - 启用 Flexbox
  • flex-direction - 主轴方向 (row, column)
  • justify-content - 主轴对齐 (center, space-between, space-around)
  • align-items - 交叉轴对齐 (center, flex-start, flex-end)
  • flex-wrap - 是否换行 (wrap, nowrap)
  • flex-grow - 项目放大比例
  • flex-shrink - 项目缩小比例
  • gap - 项目间距
