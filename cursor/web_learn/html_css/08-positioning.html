CSS 定位 (Position) 案例

1. Static (静态定位 - 默认)

position: static - 元素按照正常文档流排列，无法使用 top, right, bottom, left 属性。

2. Relative (相对定位)

position: relative - 相对于自身原始位置进行偏移，仍占据原空间。

Relative 定位（未偏移）
Relative 定位（top: 20px, left: 30px）
后面的元素不受影响

3. Absolute (绝对定位)

position: absolute - 相对于最近的已定位祖先元素（relative/absolute/fixed），如果没有则相对于 body。

Absolute 定位（右上角）
Absolute 居中

4. Fixed (固定定位)

position: fixed - 相对于浏览器窗口定位，滚动页面时位置不变。

滚动页面，右下角的按钮会保持固定位置。

固定按钮

5. Sticky (粘性定位)

position: sticky - 在滚动到指定位置前表现为 relative，之后表现为 fixed。

6. Z-index (层叠顺序)

z-index - 控制定位元素的层叠顺序，数值越大越在上层。

定位属性总结

  • static - 默认值，正常文档流
  • relative - 相对定位，相对于自身
  • absolute - 绝对定位，相对于已定位祖先
  • fixed - 固定定位，相对于视口
  • sticky - 粘性定位，relative 和 fixed 的结合
  • top, right, bottom, left - 偏移属性
  • z-index - 层叠顺序
