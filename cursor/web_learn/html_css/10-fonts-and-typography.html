字体和排版案例

1. 字体族 (font-family)

Serif 字体： Times New Roman - 衬线字体，适合正式文档
Sans-serif 字体： Arial - 无衬线字体，现代简洁
Monospace 字体： Courier New - 等宽字体，适合代码
Google 字体： Roboto - 从 Google Fonts 引入的字体

2. 字体大小 (font-size)

小号字体 (0.875rem / 14px)
基础字体 (1rem / 16px)
大号字体 (1.5rem / 24px)
超大字体 (2rem / 32px)

3. 字体粗细 (font-weight)

细体 (300)
正常 (400)
粗体 (700)

4. 字体样式 (font-style)

正常样式
斜体 (italic)
倾斜 (oblique)

5. 文本装饰 (text-decoration)

无装饰
下划线
删除线

6. 行高 (line-height)

紧密行高 (1.2)： 这是紧密行高的示例文本。行与行之间的距离较小，适合标题或短文本。
正常行高 (1.6)： 这是正常行高的示例文本。行与行之间的距离适中，适合正文阅读。
宽松行高 (2.5)： 这是宽松行高的示例文本。行与行之间的距离较大，适合需要更多空间的排版。

7. 字间距 (letter-spacing)

紧密字间距 (-1px)
正常字间距 (0)
宽松字间距 (3px)

8. 字体简写 (font shorthand)

font 简写： font: italic bold 1.2rem/1.6 'Roboto', sans-serif;
包含：style weight size/line-height family

9. 文本对齐和排版

左对齐： 文本靠左对齐，这是默认的对齐方式。
居中对齐： 文本居中显示。
右对齐： 文本靠右对齐。
两端对齐： 文本在左右两边都对齐，通过调整单词之间的间距来实现。这种对齐方式常用于书籍和正式文档中。

字体和排版属性总结

  • font-family - 字体族
  • font-size - 字体大小 (px, em, rem, %)
  • font-weight - 字体粗细 (100-900, normal, bold)
  • font-style - 字体样式 (normal, italic, oblique)
  • text-decoration - 文本装饰 (none, underline, line-through)
  • line-height - 行高 (数字, px, em, %)
  • letter-spacing - 字间距
  • text-align - 文本对齐 (left, center, right, justify)
  • font - 字体简写属性
