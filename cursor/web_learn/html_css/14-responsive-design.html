6. 媒体查询断点演示

调整窗口大小，观察背景色变化：

  • 超大屏 (>1200px): 黄色
  • 大屏 (992px-1200px): 蓝色
  • 中屏 (768px-992px): 绿色
  • 小屏 (576px-768px): 红色
  • 超小屏 (<576px): 灰色

响应式设计要点

  • viewport 元标签 - 设置移动端视口
  • 媒体查询 - @media 定义断点
  • 弹性单位 - 使用 %, em, rem, vw, vh
  • 弹性图片 - max-width: 100%, height: auto
  • 移动优先 - 先设计移动端，再扩展
  • 常用断点 - 576px, 768px, 992px, 1200px
