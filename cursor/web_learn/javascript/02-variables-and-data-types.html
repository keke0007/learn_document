变量和数据类型

1. 变量声明（var, let, const）

// var - 函数作用域，可以重复声明 var name = "张三"; var name = "李四"; // 可以重复声明 // let - 块作用域，不能重复声明 let age = 25; // let age = 30; // 错误：不能重复声明 // const - 块作用域，常量，不能重新赋值 const PI = 3.14159; // PI = 3.14; // 错误：不能重新赋值

2. JavaScript 数据类型

Number（数字）
String（字符串）
Boolean（布尔值）
Undefined（未定义）
Null（空值）
Object（对象）
Symbol（符号）
BigInt（大整数）

3. Number（数字类型）

let integer = 42; // 整数 let decimal = 3.14; // 小数 let negative = -10; // 负数 let scientific = 1.5e3; // 科学计数法 (1500) let infinity = Infinity; // 无穷大 let notANumber = NaN; // 非数字

4. String（字符串类型）

let singleQuote = '单引号字符串'; let doubleQuote = "双引号字符串"; let templateLiteral = `模板字符串 ${variable}`; // 字符串方法 let str = "Hello World"; str.length; // 长度 str.toUpperCase(); // 转大写 str.toLowerCase(); // 转小写 str.charAt(0); // 获取字符

5. Boolean（布尔类型）

let isTrue = true; let isFalse = false; // 布尔转换 Boolean(1); // true Boolean(0); // false Boolean(""); // false Boolean("text"); // true Boolean(null); // false Boolean(undefined); // false

6. Undefined 和 Null

let undefinedVar; // undefined - 未赋值的变量 let nullVar = null; // null - 空值（有意的空值） // 区别 undefined == null; // true（值相等） undefined === null; // false（类型不同）

7. Object（对象类型）

// 对象字面量 let person = { name: "张三", age: 25, city: "北京" }; // 访问属性 person.name; // "张三" person["age"]; // 25 // 添加属性 person.email = "zhang@example.com";

8. 类型检测

typeof 42; // "number" typeof "hello"; // "string" typeof true; // "boolean" typeof undefined; // "undefined" typeof null; // "object" (历史遗留问题) typeof {}; // "object" typeof []; // "object" // 更好的类型检测 Array.isArray([]); // true instanceof Array; // 检查实例

9. 类型转换

// 转换为字符串 String(123); // "123" (123).toString(); // "123" // 转换为数字 Number("123"); // 123 parseInt("123"); // 123 parseFloat("3.14"); // 3.14 // 转换为布尔值 Boolean(1); // true Boolean(0); // false
