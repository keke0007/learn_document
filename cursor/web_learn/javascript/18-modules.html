模块化

注意：ES6 模块需要在服务器环境下运行，不能直接通过 file:// 协议打开。 可以使用本地服务器（如 Live Server）或查看代码示例。

1. 导出（export）

// math.js - 命名导出 export function add(a, b) { return a + b; } export function multiply(a, b) { return a * b; } export const PI = 3.14159; // 或者使用 export { } const subtract = (a, b) => a - b; export { subtract }; // 默认导出 export default function divide(a, b) { return a / b; }

2. 导入（import）

// 导入命名导出 import { add, multiply, PI } from './math.js'; // 导入默认导出 import divide from './math.js'; // 导入所有命名导出 import * as math from './math.js'; math.add(1, 2); // 重命名导入 import { add as sum } from './math.js'; // 同时导入默认和命名 import divide, { add, multiply } from './math.js';

3. 模块示例结构

// utils.js export function formatDate(date) { return date.toLocaleDateString(); } export function formatCurrency(amount) { return `¥${amount.toFixed(2)}`; } // user.js export class User { constructor(name, email) { this.name = name; this.email = email; } greet() { return `Hello, ${this.name}!`; } } // main.js import { formatDate, formatCurrency } from './utils.js'; import { User } from './user.js'; const user = new User("张三", "zhang@example.com"); console.log(user.greet());

4. 动态导入

// 动态导入（返回 Promise） async function loadModule() { const module = await import('./math.js'); console.log(module.add(1, 2)); } // 条件导入 if (condition) { import('./moduleA.js').then(module => { // 使用模块 }); }

5. HTML 中使用模块

<script type="module"> import { add } from './math.js'; console.log(add(1, 2)); </script> <!-- 或者外部模块 --> <script type="module" src="main.js"></script>
