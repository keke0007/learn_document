3. 语句和分号

// JavaScript 语句以分号结尾（可选，但建议使用） let name = "张三"; let age = 25; // 也可以不使用分号（不推荐） let city = "北京"

4. 代码块

// 使用花括号 {} 定义代码块 { let x = 10; let y = 20; console.log(x + y); } // 函数代码块 function greet() { console.log("Hello!"); }

5. 标识符命名规则

// 合法标识符 let userName = "张三"; // 驼峰命名 let user_age = 25; // 下划线命名 let $price = 100; // 可以以 $ 开头 let _count = 5; // 可以以 _ 开头 // 不合法标识符 // let 123name = "test"; // 不能以数字开头 // let user-name = "test"; // 不能包含连字符 // let let = "test"; // 不能使用关键字

6. 关键字和保留字

JavaScript 关键字：if, else, for, while, function, var, let, const, return, class, import, export 等

这些词不能用作变量名、函数名或其他标识符

7. 严格模式

// 在脚本开头使用 "use strict" 启用严格模式 "use strict"; // 严格模式下，未声明的变量会报错 // x = 10; // 错误：x is not defined // 必须使用 let, const 或 var 声明变量 let x = 10; // 正确
